# EI-report-parser

這是一個專利為「兒童職能治療評估報告」設計的自動化工具。它可以將 `.docx` 檔案內容拆解為結構化的區塊，便利治療師快速查閱與複製內容。

## 📋 功能說明
- **自動化解析**：將報告拆解為精細動作、感覺統合、日常生活自理等分類。
- **一鍵複製**：所有分類區塊皆支援點擊即複製，方便填寫。
- **檔案隱私**：檔案僅在記憶體中暫存解析，完成後立即刪除。

## 🚀 啟動方法
1. 安裝必要套件：`pip install -r requirements.txt`
2. 執行伺服器：`python server.py`
3. 開啟網頁：`http://localhost:5001`

## ☁️ 部署
本專案已完成 Vercel 適配。直接將專案同步到 GitHub 並在 Vercel 匯入即可部署。

## 📊 解析結構
工具會自動提取並分類以下項目：

```
基本資訊
├── 姓名、性別、評估日期等

家屬主訴與期待
├── 家長的擔心與期待

職能評估
├── 精細動作
│   ├── 評估工具
│   └── 行為觀察及綜合結果
├── 感覺統合
│   ├── 評估工具
│   └── 行為觀察及綜合結果
├── 日常生活自理
│   ├── 飲食
│   ├── 穿脫衣
│   ├── 盥洗衛生
│   ├── 遊戲活動
│   └── 生活作息及參與
└── 其他（認知發展）

問題分析
├── 各項問題的分析

總結與建議
├── 精細動作部分
├── 認知發展
├── 感覺統合部分
└── 人際互動部分
```
